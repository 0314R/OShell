
The grammar accepts a SCRIPT as well as a composition as an input line. 
So any script filename that is located in the current directory and 
called in the shell is executed 

When the command red is entered in the terminal, the color is changed to red. 
When the command reset is entered, the color is reset. We also included other
colors, whose functions can be found in the colors.h and colors.c files.
Something to take into account is that the user needs to know what commands
are used for which colors. The commands for red, or blue for example make sense
but the ones for light colors are less intuitive,
